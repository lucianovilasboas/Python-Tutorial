# 📓 Tutorial Completo de Jupyter Notebook

**Do básico ao avançado — aprenda a usar o Jupyter como um profissional**

---

## 📖 Apresentação

Bem-vindo ao **Tutorial Completo de Jupyter Notebook**! Este material foi
criado para ensinar tudo o que você precisa saber sobre o Jupyter, desde a
instalação até boas práticas de compartilhamento. São 16 capítulos em
português, com exemplos práticos e exercícios, perfeitos para estudantes,
professores e profissionais de dados.

## 🎯 Objetivos
- Ensinar o que é Jupyter e como instalá-lo no seu computador
- Dominar a interface, o dashboard e os tipos de célula
- Escrever células de código e Markdown com formatação avançada
- Aprender atalhos de teclado e técnicas de produtividade
- Entender o funcionamento do kernel e como gerenciá-lo
- Criar análises de dados com Pandas e visualizações com Matplotlib
- Usar widgets interativos e comandos mágicos
- Explorar JupyterLab vs Notebook Clássico e extensões (Nbextensions)
- Compartilhar seus notebooks e seguir boas práticas

## ✅ Pré-requisitos
- Computador com Python 3 instalado (recomendado: 3.10 ou superior)
- Navegador moderno (Chrome, Firefox, Edge)
- Vontade de aprender e explorar

## 📚 Como Utilizar Este Material
1. Ordem sugerida: Siga os capítulos na ordem numérica (J00 a J15)
2. Leia atentamente cada explicação antes de executar os exemplos
3. Execute as células no seu próprio ambiente Jupyter
4. Experimente modificar os exemplos para fixar o aprendizado
5. Pratique os atalhos e comandos mágicos no seu dia a dia

> Dica: Jupyter é uma ferramenta poderosa — quanto mais você explorar,
> mais descobrirá!

## 📌 Dicas de Estudo
- **Pratique os atalhos**: Modo comando (Esc) vs modo edição (Enter) — use diariamente
- **Experimente kernels**: Teste diferentes kernels para ver como cada linguagem se comporta
- **Use Markdown**: Documente seus notebooks como se fosse um caderno de laboratório
- **Explore comandos mágicos**: `%lsmagic` mostra todos os comandos disponíveis
- **Salve com frequência**: Ctrl+S é seu melhor amigo, mas os checkpoints automáticos ajudam
- **Compartilhe**: Use nbviewer ou GitHub para mostrar seu trabalho
